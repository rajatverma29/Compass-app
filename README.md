# Compass App with Dial UI

A modern Android Compass application built using Kotlin and XML with real-time orientation tracking and a custom dial-based interface. The app uses device sensors to provide smooth, accurate, and responsive directional updates.

## Features

* Real-time orientation tracking
* Custom dial-based compass UI
* Smooth and responsive performance
* Lightweight and user-friendly design

## Tech Stack

* Kotlin
* XML
* Android Studio
* Device Sensors
* SVG Assets
